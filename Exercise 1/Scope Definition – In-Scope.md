## Scope Definition – In-Scope

The following features and functionalities are included within the scope of the **Hackathon Team Management System**. These define what the system is expected to deliver as part of the project.

---

### In-Scope Features

- User registration and login functionality  
- Team creation and management by team leaders  
- Invitation of members to teams  
- Role assignment within teams  
- Viewing and managing team details  
- Basic user profile management  
- Centralized platform for team coordination  
- Secure access to system features  
- Basic validation and error handling  

---

### In-Scope Activities

- Requirement analysis and documentation  
- System design and architecture planning  
- Development of core system functionalities  
- Functional and basic usability testing  
- Project documentation and reporting  

---

> ✅ **Note:**  
> Any features or enhancements not explicitly mentioned above are considered outside the scope of the current project phase.
