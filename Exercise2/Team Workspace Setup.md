## Team Workspace Setup

This section describes the setup of the team workspace for effective collaboration using the selected CASE tool.

---

### Workspace Creation

- A shared project workspace was created in **StarUML / Visual Paradigm** for the **Hackathon Team Management System**.
- The project file was saved with a standardized naming format.
- The workspace was stored in a shared folder accessible to all team members.

---

### Folder Structure

The following structure was maintained for organization:

- /Diagrams  
  - Use Case Diagrams  
  - Class Diagrams  
  - Sequence Diagrams  
  - Activity Diagrams  

- /Documentation  
- /Reports  

---

### Collaboration Guidelines

- Only assigned members modify specific diagrams to avoid conflicts.  
- Updated versions are shared immediately after changes.  
- File naming conventions are followed to maintain version clarity.
