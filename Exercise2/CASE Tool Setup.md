## CASE Tool Setup

This section describes the installation and configuration of a CASE (Computer-Aided Software Engineering) tool used for system modeling and diagram creation.

---

### Selected Tool
**StarUML** (or **Visual Paradigm Community Edition**)

---

### Installation Steps

1. Visit the official website of the selected CASE tool:
   - StarUML: https://staruml.io  
   - Visual Paradigm: https://www.visual-paradigm.com  

2. Download the appropriate installer based on your operating system (Windows / macOS / Linux).

3. Run the installer and follow the on-screen instructions to complete the installation.

---

### Configuration Steps

- Launch the installed CASE tool.  
- Create a new project file for the **Hackathon Team Management System**.  
- Set up the project workspace and save the project in the designated folder.  
- Enable required diagram types such as:
  - Use Case Diagram  
  - Class Diagram  
  - Sequence Diagram  
  - Activity Diagram  

---

### Purpose of Using CASE Tool

The CASE tool is used to:
- Design and visualize system architecture  
- Create UML diagrams  
- Improve clarity of system requirements  
- Support proper documentation for SEAD evaluation
