## Actors Identification

This section identifies all the actors that interact with the **Hackathon Team Management System**. Actors represent users or external systems that interact with the application.

---

### Primary Actors

| Actor | Description |
|-------|------------|
| Student | Registers, joins teams, views team details, and participates in hackathon activities. |
| Team Leader | Creates teams, invites members, assigns roles, and manages team information. |
| Admin | Manages overall system activities, monitors teams, and oversees platform usage. |

---

### Secondary Actors

| Actor | Description |
|-------|------------|
| Hackathon Organizer | Observes team formation and ensures smooth event coordination. |
| System | Handles authentication, data storage, validation, and automated processes internally. |
